# S3 Static Website (Terraform)

This directory provisions a minimal S3 static website using Terraform.

Defaults chosen:
- Region: inherit from parent/provider configuration (do not set here)
- Bucket name: autogenerated unless you set `bucket_name`
- Public website: enabled via bucket policy (HTTP only)
- `force_destroy`: false by default

Quick start (PowerShell):

```powershell
cd "06-s3-static website"
terraform init
terraform plan -out plan.tfplan
terraform apply "plan.tfplan"
```

After apply, outputs include `website_url` and `bucket_name`.

If you want HTTPS or a custom domain, consider adding CloudFront and Route53.
